# Barber Shop Appointment Booking System

A web-based barber shop appointment booking system developed using Python and Flask.  
This application helps manage customer appointments digitally and reduces manual booking efforts.

## Features
- Customer registration and login
- Appointment booking with date and time selection
- Appointment confirmation page
- Admin view to manage booked appointments
- User-friendly interface using HTML and CSS

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## Project Structure
- app.py – Main Flask application file
- templates/ – Contains all HTML templates
- static/ – Contains CSS files (if applicable)

## How to Run the Project
1. Install Python
2. Install Flask using `pip install flask`
3. Run `app.py`
4. Open browser and go to `http://localhost:5000`
